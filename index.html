React Native

React Native is an open-source UI software framework created by Meta Platforms

It is used to develop applications for Android, Android TV, iOS, macOS, tvOS, Web,Windows and others by enabling developers to use the React framework along with native platform capabilities. It is also being used to develop virtual reality applications at Oculus.

React Native brings React's declarative UI framework to iOS and Android. With React Native, you use native UI controls and have full access to the native platform.

  • Declarative
  • Component-Based
  • Developer Velocity
  • Portability

React Native is developed and supported by many companies and individual core contributors

Examples of apps built with React Native

  • Facebook/Instagram
  • Skype
  • Walmart
  • Shine
  • UberEats
  • Pinterest
